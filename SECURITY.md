# Security Policy

## Supported Versions

TRIT open core is under active development. We support only the latest release for security fixes during pre-1.0.

## Reporting a Vulnerability

Please email security@trinaryvm.org with details and a proof of concept if possible. We will acknowledge receipt within 72 hours and provide a remediation timeline once triaged.

## Phased Hardening Plan

1. Internal review and threat modeling of critical components.
2. Low-cost community bug bounty focused on core surfaces.
3. Partner-funded third-party audit for high-risk components.
4. Ongoing bounty and periodic audits as the surface area grows.

## Disclosure Policy

- We follow responsible disclosure. Please do not file public issues for suspected vulnerabilities.
- We will credit reporters in release notes unless anonymity is requested.
